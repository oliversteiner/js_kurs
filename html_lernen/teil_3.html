Um einen Umbruch zu erzwingen nutze das "br"-TAG.

Ein paar Begriffe

  • Die Dinger links und recht von den Texten heissen TAGs
  • TAGs kommen meistens in Paaren vor: ein Anfangstag und ein Schlusstag.
  • TAGs dürfen nicht verschachtelt werden.

Es gibt ein paar wenige TAGs ohne Schluss:

  1. hr - Zeichnet eine Linie
  2. br - Manueller Umbruch
  3. img - Zeigt ein Bild an

Im Start-TAG können weitere information hinterlegt werden.

Diese heissen Attribute und werden folgenderweise geschrieben: name="wert"

Jedes TAG kann folgene Attribute haben:

  1. id - Eine eindeutige Bezeichnung die nur einmal pro Seite vorkommen darf
  2. class - eine Allgemeine Bezeichnung, die mehrfach vorkommen darf
